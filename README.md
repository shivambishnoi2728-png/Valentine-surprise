## üîß Troubleshooting

**Photos not showing?**
- Make sure photo files are in the same folder as HTML files
- Check that filenames match exactly (photo1.jpg, photo2.jpg, etc.)
- Make sure file extensions are lowercase (.jpg not .JPG)

**GIFs not loading?**
- Check your internet connection
- Try different GIF URLs from Giphy/Tenor
- Make sure you copied the full URL

**NO button not disappearing on mobile?**
- This is normal! On mobile it still moves when tapped
- The touch interaction works differently but still fun!

**Site looks weird on phone?**
- The site is responsive and should work on all devices
- Clear your browser cache and reload
